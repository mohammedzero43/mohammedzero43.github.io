Game Development Tools

CSG Mesh Exporter


The built-in CSG Meshes in Godot Engine is a powerful tool, however it lacks the ability to export the meshes, so CSG Mesh Exporter can become handy to allow users to export their CSG models with the applied materials.

BGE sprite animator


Since Blender Game Engine doesn't have a proper sprite controller, this script gives BGE users full control over game sprites, including sprite speed, frame, and sub-images.
